###Inner folder structure

- HEAD
- index
- objects/
    - real files copies (blob)
    - trees
    - commits
- refs/
    - references to commits

###Commands list

`init <folder>`

`add <file>`

`commit <message>`

`branch [-d] <branch-name>`

`checkout (<commit-hash> | <branch-name>)`

`log`

`merge <branch-name>`
